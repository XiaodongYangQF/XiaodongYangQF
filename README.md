## Selected Projects

- **Option-Implied Density Estimation**  
  Recovering risk-neutral densities from index options using model-free quantiles, GEV/GPD tails, and direct SPD methods.

- **Density Forecasting and Risk Management**  
  Evaluating predictive densities using log scores, CRPS, PIT tests, VaR, and ES backtests.

- **Machine Learning for Financial Prediction**  
  Applying feature selection, clustering, and deep learning models to financial market forecasting.
